# 노래 음성 변환 (SVC) — 향후 구현 예정

## 개요
영어 노래를 한국어로 부르게 하거나, A의 목소리로 B의 노래를 부르게 하는 기능.
현재 TTS(말하기)와는 다른 기술이므로 별도 모드로 구현.

## 기술
- **RVC (Retrieval-based Voice Conversion)**: 음색 변환, 가벼움 (~200MB)
- **So-VITS-SVC**: 높은 품질, 학습 필요
- **DiffSinger**: 악보 기반 합성

## 워크플로우 예상
```
1. 원곡 드롭 (영어 노래)
2. AudioForge 음악 분리로 보컬 추출
3. 참조 음성 등록 (목표 화자)
4. 언어 선택 (한국어)
5. 가사 입력 (한국어 가사)
6. SVC 모델이 원곡의 멜로디 + 리듬을 유지하면서 목소리/언어 변환
7. 변환된 보컬 + 원곡 반주(MR) 합성
```

## 필요 모델
| 모델 | 용도 | 크기 |
|------|------|------|
| RVC v2 | 음색 변환 | ~200MB |
| Whisper | 원곡 가사 추출 | 이미 설치됨 |
| NLLB-200 | 가사 번역 | 이미 설치됨 |
| Demucs | 보컬/MR 분리 | 이미 설치됨 |

## AudioForge 연계
이미 구현된 기능들과 자연스럽게 연결:
- Demucs로 보컬 분리 → SVC 입력
- Whisper로 가사 추출 → 번역
- 노래방 모드(MR 재생) + 변환된 보컬 합성

## 우선순위
현재 TTS(음성 합성) 완료 후 진행
